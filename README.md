
# Handwritten Digit Recognition using CNN

## Project Description
This project uses a Convolutional Neural Network (CNN) to classify handwritten digits (0–9) using the MNIST dataset.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Features
- Data preprocessing (normalization, reshaping)
- CNN model with convolution and pooling layers
- Model training and evaluation
- Prediction visualization

## Output
The model predicts handwritten digits with good accuracy and displays predicted vs actual results.
